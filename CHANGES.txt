=== 0.9.1 (2026-01-25) ===

This version adds the missing font file, better multi-monitor and
multi-resolution handling, and a couple of functions to the query language.
And it avoids a really obscure segfault(!) from the low-level audio system.

General:
- [fix] Fix very obscure but frequent crash caused by audio callbacks

Library:
- [new] Optional Playlist column for tracks views
- [new] Query attribute #playlist
- [new] Query functions #min, #max, #avg, #if
- [new] Playlist column for Repair log
- [fix] Export with position starts with position 01
- [fix] When rescanning playlist track, propagate result to library
- [fix] When applying undo to library playlist, also update in-memory copy

UI:
- [new] Abort any mouse drag action with click of right button
- [new] Store window size and position as proportional to screen size
- [new] Automatically adjust window size when dragged to different screen
- [chg] Glue view dividers to nearest border on window resize
- [fix] Correctly restore view dividers on start-up

Build:
- [fix] Add missing font file to repo
- [chg] Add .opam file (thanks to kit-ty-kate)
- [chg] Remove obsolete auxiliary opam repo for Raylib 1.6.0
  (if you built 0.9.0 before, you may have to run
   `opam repo remove opam-raylib-1.6.0` and `opam update -u` once)
